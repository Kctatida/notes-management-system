# Notes Management System

Это простое fullstack приложение для управления заметками.

---

## 📌 Возможности

- Создание заметок
- Просмотр заметок
- Редактирование заметок
- Удаление заметок

---

## 🛠 Технологии

**Backend:**
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server

**Frontend:**
- React (Vite)
- TypeScript
- Axios

---

## 🚀 Запуск

### Backend
```bash
cd BoardNoteAPI
dotnet run
```


### Frontend
```bash
cd frontend
npm install
npm run dev
